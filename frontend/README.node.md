# Running with Node (production)

This project uses Vite and React (TypeScript). To run the built app with Node (Express):

1. Install dependencies

```powershell
npm install
```

2. Build the app

```powershell
npm run build
```

3. Start the Node server

```powershell
npm start
```

The app will be available at http://localhost:3000 by default.

For development use `npm run dev` (Vite dev server).
